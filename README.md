# Graph Bridges Finder (JavaFX)

**Graph Bridges Finder** — графическое JavaFX-приложение для визуального создания, редактирования и анализа графов. Основная задача — **поиск мостов (bridges) в графе** с наглядной визуализацией.

---

##  Основные возможности

### Визуальное редактирование графов
- Добавление и удаление вершин щелчком мыши  
- Добавление и удаление рёбер  
- Перетаскивание вершин мышью  
- Изменение размера и положения элементов  

### Алгоритмические операции
- **Поиск мостов** в графе с пошаговой визуализацией  
- Быстрое получение результата без анимации  

### Работа с файлами
- Загрузка графов из текстовых файлов  
- Сохранение графов и результатов работы алгоритма  

### Режимы работы
- **Редактирование** — добавление, удаление и перемещение элементов  
- **Выполнение алгоритма (Run)** — пошаговая визуализация поиска мостов  
- **Быстрое получение результата (Get Result)** — мгновенный вывод результата  





# Запуск через maven:
### mvn clean install
### mvn clean javafx:run
